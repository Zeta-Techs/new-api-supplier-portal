## Requirements
### Requirement: Responsive Container Width
The UI SHALL be responsive and size the main container relative to the viewport.

#### Scenario: Desktop width
- **WHEN** the viewport width is above the mobile breakpoint
- **THEN** the main container occupies approximately 90% of the viewport width

#### Scenario: Narrow screens
- **WHEN** the viewport width is below the mobile breakpoint
- **THEN** the main container occupies approximately 98% of the viewport width
